Politica de privacidade

Coletamos dados de treino do Strava apenas para gerar analises e stickers.

Seus dados nao sao vendidos e podem ser removidos a qualquer momento.

Utilizamos a API oficial do WhatsApp para entregar mensagens e arquivos.

Para excluir seus dados, envie a mensagem \"inmoovi:delete\" no WhatsApp.
